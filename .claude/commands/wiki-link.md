# /wiki-link — เชื่อมโยงคลังความรู้ (LLM Wiki)

เป้าหมาย: ทำให้ vault เป็น graph ที่เชื่อมกัน ไม่ใช่โน้ตกระจัดกระจาย (สไตล์ Karpathy LLM Wiki)

## ขั้นตอน
1. **อ่านทั้ง vault** — ไล่ `vault/10-Concepts/`, `vault/20-Protocols/`, `vault/30-Sources/`
2. **หา missing links** — โน้ตไหนพูดถึงแนวคิดที่มีโน้ตอยู่แล้วแต่ยังไม่ได้ `[[link]]` → เติมให้
3. **หา orphan notes** — โน้ตที่ไม่มีใครลิงก์หาเลย → รายงาน + เสนอว่าควรเชื่อมกับอะไร
4. **อัปเดต MOC** — เขียน/ปรับ `vault/00-Index/MOC-Sleep.md`:
   - จัดกลุ่มโน้ตตามธีม (เช่น Circadian, Sleep Architecture, สาร/ฮอร์โมน, Protocols)
   - ลิงก์ `[[...]]` ไปโน้ตสำคัญของแต่ละธีม
   - เขียนคำอธิบายสั้นๆ ให้อ่านแล้วเห็นภาพรวมได้
5. **เสนอโน้ตที่ควรมีแต่ยังไม่มี** — ช่องว่างความรู้ (เช่น มี Adenosine แต่ยังไม่มี Caffeine)

## กติกา
- อย่าสร้างลิงก์มั่ว — ลิงก์เฉพาะที่มีความสัมพันธ์เชิงเนื้อหาจริง
- รายงานสรุปตอนจบว่าเติมลิงก์กี่จุด, orphan เหลือกี่โน้ต, เสนอโน้ตใหม่อะไรบ้าง
