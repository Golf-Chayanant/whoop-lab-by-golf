# /advise — ขอคำแนะนำเรื่องการนอน

คำถาม: $ARGUMENTS

## ขั้นตอน
1. **ค้น vault** — หาโน้ตใน `vault/` ที่เกี่ยวกับคำถาม (concepts + protocols + sources)
2. **ตอบโดยยึดจากคลังความรู้** — สังเคราะห์คำตอบจากโน้ตที่เจอ
3. **อ้างอิงเสมอ** — ระบุ `[[โน้ต]]` ที่ใช้ประกอบคำตอบ เพื่อให้ตรวจสอบย้อนได้
4. **ถ้าคลังไม่มีข้อมูล** — บอกตรงๆ ว่า "ยังไม่มีในคลังความรู้" และเสนอว่าควร ingest
   คลิปหัวข้อไหนเพิ่ม อย่าเดา/แต่งข้อมูลเอง

## รูปแบบคำตอบ
- **สรุปคำแนะนำ** (ตอบตรงคำถามก่อน)
- **เหตุผล/กลไก** (อ้าง `[[concept]]`)
- **วิธีทำจริง** (อ้าง `[[protocol]]` — ทำอะไร เมื่อไหร่ นานแค่ไหน)
- **ที่มา** (ลิงก์ `[[source]]`)

## กติกา
- ภาษาไทยเป็นหลัก คงศัพท์เทคนิคอังกฤษ
- ถ้าเป็นปัญหาสุขภาพจริงจัง (นอนไม่หลับเรื้อรัง, sleep apnea) เตือนให้ปรึกษาแพทย์
