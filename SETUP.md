คู่มือนี้พาคุณจาก "โหลดโฟลเดอร์มา" → "มีคลังความรู้การนอนของตัวเอง" ทีละขั้น
ไม่ต้องเขียนโค้ดเป็น

---

## 1. สิ่งที่ต้องมี

| โปรแกรม | ใช้ทำอะไร | จำเป็นไหม |
|---------|-----------|-----------|
| **Claude Code** | ตัวหลัก — คุยกับ Claude สั่ง `/ingest` ฯลฯ | ✅ จำเป็น |
| **NotebookLM** (เว็บ ฟรี) | ดึง transcript จากคลิป YouTube | ✅ จำเป็น |
| **Obsidian** (ฟรี) | เปิดดูคลังความรู้เป็น graph สวยๆ | ⭕ ไม่บังคับ (แต่แนะนำ) |

ติดตั้ง Claude Code: ดู https://claude.com/claude-code

---

## 2. เอาเทมเพลตมาลงเครื่อง

**ถ้ามี git:**
```bash
git clone <URL-ของ-repo-นี้> sleep-expert
cd sleep-expert
```

**ถ้าไม่มี git:** กดปุ่มเขียว **Code → Download ZIP** บน GitHub → แตกไฟล์ → เปิด Terminal เข้าโฟลเดอร์นั้น

---

## 3. เปิด Claude Code

ใน Terminal ที่อยู่ในโฟลเดอร์ `sleep-expert` พิมพ์:
```bash
claude
```
Claude จะอ่าน `CLAUDE.md` อัตโนมัติ และรู้ตัวว่าเป็น "ผู้เชี่ยวชาญเรื่องการนอน" ทันที
ลองพิมพ์ `/` จะเห็นคำสั่ง `/ingest` `/wiki-link` `/advise` โผล่มา

---

## 4. เอาความรู้เข้าคลัง (ครั้งแรก)

1. เปิด https://notebooklm.google.com → **New notebook** → **Add source → YouTube** → วาง link คลิป
   การนอนที่สนใจ (แนะนำ: Matthew Walker, Andrew Huberman)
2. เปิด transcript ที่ NotebookLM สร้าง → กด **Ctrl/Cmd + A** เลือกทั้งหมด → **copy**
3. กลับมาที่ Claude Code → **วาง transcript** ลงไป แล้วพิมพ์:
   ```
   /ingest
   ```
4. Claude จะ clean + สรุปเป็นภาษาไทย + สร้างโน้ต atomic ใน `vault/` พร้อม `[[ลิงก์]]` ให้เอง
   (อาจถามชื่อคลิป / ผู้พูด / URL เพิ่ม — ตอบไปได้เลย)

> 💡 อยากได้คุณภาพดี: **วาง transcript ดิบยาวๆ ตรงๆ อย่าสรุปเองก่อน** — Claude ต้องเห็นตัวเลข
> และรายละเอียดครบเพื่อกลั่นโน้ตให้แม่น

ทำซ้ำข้อ 4 กับหลายๆ คลิป คลังจะโตขึ้นเรื่อยๆ

---

## 5. เชื่อมโยงความรู้

พอมีโน้ตหลายใบ พิมพ์:
```
/wiki-link
```
Claude จะหาความเชื่อมโยงระหว่างโน้ต เติม `[[ลิงก์]]` ที่ขาด และอัปเดตหน้าสารบัญ `MOC-Sleep.md`

---

## 6. ขอคำแนะนำ

ถามอะไรก็ได้เรื่องการนอน:
```
/advise ตอนเช้าควรโดนแดดกี่นาที ทำไม
```
Claude จะตอบโดย **อ้างอิงโน้ตในคลัง** (ไม่เดา) ถ้าคลังยังไม่มีข้อมูล มันจะบอกตรงๆ

---

## 7. (แนะนำ) เปิดดูใน Obsidian

1. เปิด Obsidian → **Open folder as vault** → เลือกโฟลเดอร์ **`vault`** (ไม่ใช่โฟลเดอร์นอกสุด)
2. กด **Graph View** → เห็นใยความรู้เชื่อมกัน
3. เริ่มอ่านจาก `00-Index/MOC-Sleep.md`

## เจอปัญหา?
- **ไม่เห็น slash command** → เช็คว่าเปิด `claude` จากในโฟลเดอร์เทมเพลต (ที่มี `.claude/`)
- **NotebookLM import คลิปไม่ได้** → คลิปต้องมี caption และไม่ใช่คลิปที่เพิ่งลง <72 ชม.
- **โน้ตภาษาอังกฤษ** → บอก Claude ว่า "ตอบไทย" ได้เลย หรือเช็คกฎใน `CLAUDE.md`
